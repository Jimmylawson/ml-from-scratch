[ ] Beyond accuracy, the key classiction metrics are 
Precision:
  Precision = TP / TP + FP
"which means of predicted positives, how many were correct? "

Recall :
  Recall = TP / TP + FN
"which means of actual positives, how many did we catch? "

F1 score:
  F1 = 2 * (Precision * Recall) / (Precision + Recall)
"balance between precision and recall"
